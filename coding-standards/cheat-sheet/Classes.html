Classes

D R A F T
Don'tDo
Opening brace of a class must be on the line after the definition. 
/**
 * Comment
 *
 * @package  Foo
 * @since    foo
 */
class Foo {
}
 

/**
 * Comment
 *
 * @package  Foo
 * @since    foo
 */
class Foo
{
}
When calling class constructors with no arguments, do not include parentheses. 
$foo = new Foo();
 

$foo = new Foo;
Check for spaces between classname and opening parenthesis. 
$foo = new Foo ($i);
 

$foo = new Foo($i);
Check argument formating. Space after comma. 
$foo = new Foo($i,$i);
 

$foo = new Foo($i, $i);
Usage of "$this" in static methods will cause runtime errors. 
/**
 * Comment
 *
 * @package  Foo
 * @since    foo
 */
class Foo
{
	/**
	 * Comment
	 *
	 * @return foo
	 */
	public static function bar()
	{
		echo $this->baz;
	}
}
 

/**
 * Comment
 *
 * @package  Foo
 * @since    foo
 */
class Foo
{
	/**
	 * Comment
	 *
	 * @return foo
	 */
	public static function bar()
	{
		echo self::$baz;
	}
}
