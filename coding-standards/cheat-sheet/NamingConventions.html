NamingConventions

D R A F T
Don'tDo
Public properties must *not* have a prefix. 
/**
 * Foo class
 *
 * @package  Foo
 * @since    foo
 */
class FunctionTest
{
	public $_foo = 1;
 
 

/**
 * Foo class
 *
 * @package  Foo
 * @since    foo
 */
class FunctionTest
{
	public $foo = 1;
Protected properties must *not* have a prefix. 
	protected $_foo = 1;
 
 

	protected $foo = 1;
Private properties *must* have a prefix. 
	private $foo = 1;
 
 

	private $_foo = 1;
Public functions must *not* be prefixed with an underscore. 
	/**
	 * Comment
	 *
	 * @return foo
	 */
	public function _foo()
	{
	}
 
 

	/**
	 * Comment
	 *
	 * @return foo
	 */
	public function foo()
	{
	}
Protected functions must *not* be prefixed with an underscore. 
	/**
	 * Comment
	 *
	 * @return foo
	 */
	protected function _bar()
	{
	}
 
 

	/**
	 * Comment
	 *
	 * @return foo
	 */
	protected function bar()
	{
	}
Private method *must* be prefixed with an underscore. 
	/**
	 * Comment
	 *
	 * @return foo
	 */
	private function foobar()
	{
	}
 

	/**
	 * Comment
	 *
	 * @return foo
	 */
	private function _foobar()
	{
	}
Only PHP magic methods should be prefixed with a double underscore. 
	/**
	 * Comment
	 *
	 * @return foo
	 */
	public function __bar()
	{
	}
 
 

	/**
	 * Comment
	 *
	 * @return foo
	 */
	public function bar()
	{
	}
