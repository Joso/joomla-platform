WhiteSpace

D R A F T
Don'tDo
Tabs must be used to indent lines; spaces are not allowed. 
if (true)
{
    true;
}
 

if (true)
{
	true;
}
@TODO: Concatenation operator must be surrounded by spaces. 
$i.'test';
 

$i . 'test';
@TODO: Operators must have a space before and after. 
$i=0;
$i>0;
$i+=0;
 

$i = 0;
$i > 0;
$i += 0;
@TODO: Unary operators must not have a space. 
$i --;
-- $i;
$i ++;
++ $i;
 

$i--;
--$i;
$i++;
++$i;
@TODO: Casting must have one space. 
(int)$i;
(int)  $i;
 

(int) $i;
(int) $i;
No space before semicolon.
That's a common leftover... 
$foo = 'bar' ;
 

$foo = 'bar';
Functions and classes must not contain multiple empty lines in a row.
That's a common leftover... 
/**
 * Comment
 *
 * @return foo
 */
function foo()
{
	$foo;
 
 
	$bar;
}
 

/**
 * Comment
 *
 * @return foo
 */
function foo()
{
	$foo;
 
	$bar;
}
Please end your files with an empty line.
