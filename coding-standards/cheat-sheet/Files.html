Don'tDo
No parentheses are required for statements. 
include($foo);
include_once($foo);
require($foo);
require_once($foo);
 

include $foo;
include_once $foo;
require $foo;
require_once $foo;
File is being unconditionally included; use "require" instead. 
include $foo;
 

require $foo;
File is being unconditionally included; use "require_once" instead. 
include_once $foo;
 

require_once $foo;
